Your privacy is our priority - learn how we protect it

Last updated: July 2025

🛡️ Privacy First Design

🔐 Complete Privacy: Cascade is designed to be completely private. We prioritize your privacy above all else.

This Privacy Policy describes how Cascade Task Management handles your information. Spoiler alert: We don't handle any of your information because we never see it!

💾 Local Data Storage

All your task data is stored locally on your device in your browser's storage.

✓ What this guarantees:
  • 🔒 Data never leaves your device - Tasks stored only in browser
  • 🚫 No server transmission - Task data never sent anywhere
  • ☁️ No cloud storage - We don't store information in the cloud
  • 👤 No user accounts - No registration or personal info needed
  • 🕵️ No tracking - We don't track usage or behavior
💾 Browser Storage Details:
  • 🖥️ Managed by your browser - You have full control
  • 🧹 You can clear anytime - Complete user control
  • 🛡️ Site-isolated - Not accessible to other websites
  • 📱 Device-specific - Not synchronized across devices

📁 Import/Export

The import/export feature creates JSON files locally:

  • ✓ Generated locally on your device
  • ✓ Contains only your task data
  • ✓ Never sent to our servers
  • ✓ Under your complete control

🔗 Third-Party Services

We use these external services for basic functionality:

  • Bootstrap CSS - Styling framework
  • Google Fonts - Typography
  • Web Hosting - Static file delivery

These may collect standard web data. See their privacy policies.

🔐 Data Security

Since data never leaves your device, security depends on:

  • Your device's security settings
  • Your browser's security features
  • Regular backups using export
