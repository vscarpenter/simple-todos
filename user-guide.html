‚å®Ô∏è Keyboard Shortcuts

Cascade provides comprehensive keyboard navigation for power users and accessibility:

Global Shortcuts

  • Ctrl/Cmd + N - Create new task (focus input)
  • Ctrl/Cmd + Z - Undo last action (50 operations)
  • Ctrl/Cmd + Shift + Z - Redo action
  • Ctrl/Cmd + / - Show keyboard shortcuts help
  • Escape - Close modals or cancel editing

Task Management

  • Enter - Save task when editing
  • Tab - Navigate between interactive elements
  • Space - Activate buttons and controls
  • Arrow Keys - Navigate within task lists

Board Navigation

  • Ctrl/Cmd + B - Open board selector
  • Ctrl/Cmd + Shift + N - Create new board
  • Ctrl/Cmd + M - Open hamburger menu

üíæ Data & Backup

How Your Data is Stored

Cascade stores all your tasks locally in your browser. This means:

  • Your data never leaves your device
  • No internet connection required after loading
  • Complete privacy - we can't see your tasks
  • Fast performance with instant saves

Backing Up Your Data

To create a backup:

  1. Open the hamburger menu (‚ò∞)
  2. Go to Data ‚Üí Export Data
  3. Save the downloaded JSON file somewhere safe
  4. This file contains all your boards and tasks

Restoring from Backup

To restore your data:

  1. Open the hamburger menu (‚ò∞)
  2. Go to Data ‚Üí Import Data
  3. Select your backup JSON file
  4. Your tasks will be restored automatically

‚ö†Ô∏è Important

Regular backups are recommended! Clearing browser data or reinstalling will remove your tasks.

üîß Troubleshooting

Common Issues

  • Tasks not saving: Check if your browser allows local storage
  • Interface not loading: Try refreshing the page or clearing browser cache
  • Can't drag tasks: Ensure JavaScript is enabled in your browser
  • Import not working: Make sure you're using a valid Cascade export file

Browser Requirements

Cascade works best with modern browsers that support:

  • Local Storage API
  • ES6 JavaScript modules
  • CSS Grid and Flexbox
  • Drag and Drop API
